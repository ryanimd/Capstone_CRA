# FanFave
Live Site: [FanFave](https://morning-island-58610.herokuapp.com/)

## About
FanFave is a web app that lets you search for tv shows from a third party API and write reviews about them.

## Technologies Used

### Front End
  - React
  - Javascript
  - Axios
  - CSS3/Bulma

### Back End
  - Python
  - Django
  - PostgreSQL
